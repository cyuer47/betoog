Projectinformatie

Wat is dit project?

Dit is een schoolproject voor het vak Nederlands. We hebben een betoog geschreven over het onderwerp "Digitalisering op de middelbare school moet verbeterd worden."

Het betoog bevat een inleiding, stelling, drie argumenten, een tegenargument met weerlegging en een conclusie.

Schooljaar 2025-2026
Techniek

Hoe is de site gemaakt?

De website is gebouwd met HTML, CSS en vanilla JavaScript op de frontend. De backend gebruikt Node.js met een SQLite database.
